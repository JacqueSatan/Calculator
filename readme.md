# Calculatrice

Mon objectif est de créer une calculatrice appliquant au moins les lois de distributivité et de priorité de calcul.

C'est la troisième fois que je recommence, et cette fois-ci la méthode de calcul semble adéquate pour prendre en compte la distributivité, et les priorités de calcul ne devraient pas être très compliquées à mettre en place non plus.

Le code n'est pas encore très bien commenté, mais cela viendra.

Bien entendu, l'objectif est de ne pas copier de code, ni de chercher de réponses sur internet ou ailleurs, et de n'utiliser si possible aucune bibliothèque autre que celle qui servira à l'interface graphique.

Je n'ai rien trouvé sur internet qui corresponde à ce projet autre que la calculatrice de Microsoft Windows, c'est pour ça que je mets ça en ligne, ça pourrait servir.
